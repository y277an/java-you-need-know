# java-you-need-know

这个工程是用来存放博客演示的完整代码，会逐渐丰富各种源码原理的讲解。

目录结构如下：
```
|-- Desktop  
    |-- netty-rpc 手写RPC框架
    |-- spring-ioc-mvc 手写SpringIoC和SpringMVC框架
    |-- mini-mybatis 手写Mybatis框架
```

自我批评：
1. 工程中为了省去日志依赖的配置，所以所有输出都用了控制台输出。
2. 由于在公司和在家多台终端多个版本的IDE，所以文件头等代码注释没有做到风格统一。
